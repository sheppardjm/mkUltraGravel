## v6.0 Requirements

Requirements for UI Polish + Dev Tools milestone. Each maps to roadmap phases.

### Color Consistency

- [x] **CLR-01**: Extract shared `starColors` module from duplicate definitions in RouteMap, ElevationProfile, and GravelSectors — validated, single shared module at src/lib/starColors.ts
- [x] **CLR-02**: Map polylines for 2-star and 3-star sectors use same colors as sector cards — validated via shared import
- [x] **CLR-03**: Elevation profile bands for 2-star and 3-star sectors use same colors as sector cards — validated via shared import

### Elevation Profile Labels

- [x] **ELEV-01**: Each gravel sector displays its name on the elevation profile — validated, annotation labels on all 6 sectors
- [x] **ELEV-02**: Each gravel sector displays its star rating on the elevation profile — validated, Unicode stars in label content
- [x] **ELEV-03**: Labels positioned at bottom of chart (below elevation line) — validated, position y:'end'
- [x] **ELEV-04**: Labels staggered to avoid overlap on narrow/adjacent sectors — validated, yAdjust i%2 alternation + rotation for narrow sectors

### Site Navigation

- [x] **NAV-01**: Fixed header nav visible on all pages — validated, SiteNav.astro in BaseLayout.astro
- [x] **NAV-02**: Nav links to Home, Results, and Submission pages — validated, 3 links in navLinks array
- [x] **NAV-03**: Active page visually indicated (build-time, no flash) — validated, Astro.url.pathname + aria-current="page"
- [x] **NAV-04**: Nav z-index clears grain and Escher overlays — validated, z-index 10000
