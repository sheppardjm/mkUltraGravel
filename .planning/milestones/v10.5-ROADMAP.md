## Milestone Summary

**Key Decisions:**
- overflow-hidden scoped to media container only; content div as sibling (card DOM pattern)
- 1200x675 card crops for 2x resolution at 1440px viewports
- column-fill: balance doesn't work for horizontal-scroll masonry; max-height tuning instead
- Subdomain redirect placed before path-based rules in netlify.toml
- JPEG over WebP for OG image — universal social platform compatibility
- Astro.site (not Astro.url) for OG image URLs; new URL(pathname, site) for canonicals
- set:html={JSON.stringify()} for JSON-LD injection in Astro script tags
- Deploy preview noindex handled by Netlify platform default (no config needed)

**Issues Resolved:**
- CLASSIFIED badge clipping behind card images (z-index fix)
- Escher tessellation washing out text in sectors section
- Gallery column whitespace on large viewports
- No social sharing previews when links shared (OG/Twitter tags added)
- No crawl infrastructure for search engines (sitemap + robots.txt)
- Short meta tag strings truncated in social previews (lengthened to optimal ranges)

**Issues Deferred:**
- None

**Technical Debt Incurred:**
- None
