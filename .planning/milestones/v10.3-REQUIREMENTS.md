# Requirements: MK Ultra Gravel

**Defined:** 2026-04-08
**Core Value:** Get gravel cyclists excited enough about this ride to show up on June 7, 2026.

## v10.3 Requirements

Requirements for mobile elevation profile label fix. Each maps to roadmap phases.

### Elevation Profile Mobile

- [x] **ELEV-05**: Sector name labels hidden on elevation profile below 640px viewport width
- [x] **ELEV-06**: Sector star-rating labels hidden on elevation profile below 640px viewport width
- [x] **ELEV-07**: KOM segment labels hidden on elevation profile below 640px viewport width
- [x] **ELEV-08**: Colored sector and KOM annotation bands remain visible on all viewport sizes

## Out of Scope

| Feature | Reason |
|---------|--------|
| Horizontal scroll for elevation profile | Overengineered for current need — hiding labels is sufficient |
| Abbreviated/rotated labels | Still cluttered at 375px even with shorter text |
| Tap-to-reveal labels on mobile | Adds JS complexity for marginal benefit |
