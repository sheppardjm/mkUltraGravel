<task type="auto">
  <name>Task 1: Install Chart.js and create ElevationProfile.astro</name>
  <files>package.json, src/components/ElevationProfile.astro</files>
  <action>
1. Install dependencies:
   ```bash
   npm install chart.js chartjs-plugin-annotation
   ```

2. Create `src/components/ElevationProfile.astro` with this structure:

**HTML:** A wrapper div (`.elevation-wrapper`) containing a relatively-positioned container div (`.elevation-container`) with a `<canvas id="elevation-chart">` inside.

**Style (scoped):**
- `.elevation-wrapper`: `width: 100%; margin-top: 0.5rem;`
- `.elevation-container`: `position: relative; width: 100%; height: 140px;` (REQUIRED: `position: relative` for Chart.js responsive mode)
- Media query `@media (min-width: 768px)` → `.elevation-container { height: 180px; }`
- Fixed `px` heights only — never `%` or `vh` (causes Chart.js infinite resize loop)

**Script block** (uses `await import()` pattern matching RouteMap.astro — prevents SSR `window` errors):

a. Dynamic imports — chart.js/auto and chartjs-plugin-annotation:
   ```typescript
   const { Chart } = await import('chart.js/auto');
   const { default: AnnotationPlugin } = await import('chartjs-plugin-annotation');
   Chart.register(AnnotationPlugin);  // MUST register BEFORE new Chart()
   ```

b. Parallel fetch of route-data.json and annotations.json (established Promise.all pattern):
   ```typescript
   const [routeData, annotations] = await Promise.all([
     fetch('/data/route-data.json').then(r => r.json()),
     fetch('/data/annotations.json').then(r => r.json())
   ]);
   ```

c. Map route data to `{x, y}` shape required by `parsing: false` mode:
   ```typescript
   const chartData = routeData.map((pt: { mi: number; ele: number }) => ({
     x: pt.mi,
     y: pt.ele
   }));
   ```

d. Build sector band annotation boxes from annotations.sectors — use starColors map:
   ```typescript
   const starColors: Record<number, string> = {
     1: '#888888', 2: '#aaaaaa', 3: '#f5a623', 4: '#e86d1f', 5: '#c0392b'
   };
   const annotationBoxes: Record<string, object> = {};
   annotations.sectors.forEach((sector: { name: string; startMi: number; endMi: number; stars: number }, i: number) => {
     annotationBoxes[`sector_${i}`] = {
       type: 'box',
       xMin: sector.startMi,
       xMax: sector.endMi,
       backgroundColor: starColors[sector.stars] + '22',  // 20% opacity fill
       borderColor: starColors[sector.stars] + '66',       // border
       borderWidth: 1,
     };
   });
   ```
   Note: Start with labels disabled (color band is sufficient); sector names are on the map via popup.

e. Create dark background canvas plugin:
   ```typescript
   const darkBgPlugin = {
     id: 'darkBackground',
     beforeDraw: (chart: any) => {
       const { ctx } = chart;
       ctx.save();
       ctx.globalCompositeOperation = 'destination-over';
       ctx.fillStyle = 'oklch(0.14 0.01 250)';  // matches --color-bg-surface
       ctx.fillRect(0, 0, chart.width, chart.height);
       ctx.restore();
     }
   };
   ```

f. Get canvas element with null guard:
   ```typescript
   const canvas = document.getElementById('elevation-chart') as HTMLCanvasElement;
   if (!canvas) return;
   ```

g. Create Chart instance with these options:
   - `type: 'line'`
   - Dataset: `data: chartData`, `borderColor: 'oklch(0.55 0.01 90)'`, `borderWidth: 1.5`, `backgroundColor: 'oklch(0.25 0.01 250 / 0.4)'`, `fill: true`, `tension: 0`, `pointRadius: 0`
   - `parsing: false` (data is pre-parsed {x, y})
   - `animation: false` (static chart)
   - `responsive: true`, `maintainAspectRatio: false` (REQUIRED for wrapper-controlled height)
   - Plugins: `legend: { display: false }`, `decimation: { enabled: true, algorithm: 'lttb', samples: 500 }`, `annotation: { annotations: annotationBoxes }`
   - Tooltip callbacks: label shows feet (`Math.round(ctx.parsed.y * 3.281) + ' ft'`), title shows miles (`items[0].parsed.x.toFixed(1) + ' mi'`)
   - X scale: `type: 'linear'`, `min: 0`, `max: 100`, grid color `oklch(0.25 0.01 250)`, tick color `oklch(0.55 0.01 90)`, tick callback `val + 'mi'`
   - Y scale: grid color `oklch(0.25 0.01 250)`, tick color `oklch(0.55 0.01 90)`, tick callback converts meters to feet: `Math.round(val * 3.281) + 'ft'`
   - Register `darkBgPlugin` via the `plugins: [darkBgPlugin]` array on the Chart constructor (NOT in options.plugins)

IMPORTANT: Use the complete code example from 04-RESEARCH.md as the reference implementation. The research has the exact code that should work.
