Create three new Astro components that load annotation data at build time using `fs.readFileSync`
+ `JSON.parse` in the frontmatter. All three read from `join(process.cwd(), 'public', 'data', 'annotations.json')`.

**GravelSectors.astro:**
- Import `readFileSync` from `fs` and `join` from `path`
- Load `annotations.sectors` — type: `Array<{ name: string; startMi: number; lengthMi: number; stars: number }>`
- Define `starColors` map matching RouteMap.astro exactly: `{ 1: '#888888', 2: '#aaaaaa', 3: '#f5a623', 4: '#e86d1f', 5: '#c0392b' }`
- Render each sector as a card with `classified-border bg-bg-surface p-4` classes
- Card layout: flex row with sector name (h3, `text-accent-white text-lg`) on the left, star rating on the right
- Star rating: `'★'.repeat(sector.stars) + '☆'.repeat(5 - sector.stars)` with inline `color` style from `starColors[sector.stars]`
- Star text: `text-base tracking-widest` classes for readability
- Below the name/stars row: mile marker and distance in `text-text-muted text-sm` with `flex gap-6`
- Format: "Mile {startMi}" and "{lengthMi.toFixed(1)} mi"
- Wrap all cards in a `div.space-y-4`

**KomSegments.astro:**
- Same data loading pattern as GravelSectors
- Load `annotations.kom` — type: `Array<{ name: string; startMi: number; lengthMi: number; grade: number; elevFt: number }>`
- Render each KOM as a card with `classified-border bg-bg-surface p-4` classes
- Card: h3 with `text-accent-green mb-2` for the KOM name (green = climbing accent)
- Data grid: `grid grid-cols-2 gap-x-6 gap-y-1 text-text-muted text-sm`
- Four data fields: "Mile {startMi}", "{lengthMi.toFixed(2)} mi", "{grade}% grade", "{elevFt} ft gain"
- Wrap all cards in a `div.space-y-4`

**RestockPoints.astro:**
- Same data loading pattern
- Load `annotations.restock` — type: `Array<{ name: string; mi: number }>`
- Render as a simple `ul.space-y-3` list
- Each `li`: flex layout with accent-green arrow marker (▶), stop name, and "Mile {mi}" right-aligned
- Classes: `flex items-center gap-3 text-text-body text-sm`
- Arrow: `span.text-accent-green` with `aria-hidden="true"`
- Mile marker: `span.text-text-muted.ml-auto`

IMPORTANT: Do NOT use `__dirname` (not available in ESM). Use `process.cwd()` for project root.
IMPORTANT: Do NOT attempt `import` from `public/` directory — Vite excludes it from module graph.
IMPORTANT: Headings (h3) automatically get `font-family: var(--font-display)` from the base layer — do NOT add `font-display` class.
