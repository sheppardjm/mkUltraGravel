<task type="auto">
  <name>Task 1: Separate overflow-hidden from card-hover on sector and KOM cards</name>
  <files>src/components/GravelSectors.astro, src/components/KomSegments.astro</files>
  <action>
The root cause is confirmed: `overflow-hidden` on the same element as `card-hover` clips the box-shadow (CSS Overflow Module Level 3 -- overflow:hidden clips ink overflow including box-shadow to the padding box).

The fix: wrap the card content in an inner div that gets `overflow-hidden`, while the outer div keeps `card-hover` but does NOT have `overflow-hidden`. The `overflow-hidden` is needed only to clip the cover photo image to the card boundary.

In `src/components/GravelSectors.astro` line 26, the current card div is:
```
<div class="classified-border bg-bg-surface overflow-hidden card-hover" data-reveal style={`animation-delay: ${i * 60}ms`}>
```

Change to a two-div structure:
- Outer div: `classified-border bg-bg-surface card-hover` plus `data-reveal` and `style` (NO overflow-hidden)
- Inner div wrapping all content (img + info): `overflow-hidden` only

Result for GravelSectors.astro:
```astro
<div class="classified-border bg-bg-surface card-hover" data-reveal style={`animation-delay: ${i * 60}ms`}>
  <div class="overflow-hidden">
    {sector.coverPhoto && (
      <img ... />
    )}
    <div class="p-4">
      ...existing content unchanged...
    </div>
  </div>
</div>
```

Apply the identical pattern to `src/components/KomSegments.astro` line 19:
```astro
<div class="classified-border bg-bg-surface card-hover" data-reveal style={`animation-delay: ${i * 60}ms`}>
  <div class="overflow-hidden">
    {segment.coverPhoto && (
      <img ... />
    )}
    <div class="p-4">
      ...existing content unchanged...
    </div>
  </div>
</div>
```

Do NOT change any CSS in global.css -- the `.card-hover` rules are correct. The problem is purely structural in the HTML.
Do NOT remove `overflow-hidden` entirely -- it is needed to clip cover photos to the card boundary.
Do NOT add any new classes or CSS -- just restructure the existing HTML.
  </action>
