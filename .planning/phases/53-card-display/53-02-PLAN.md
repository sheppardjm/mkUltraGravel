<objective>
Fix CLASSIFIED badge hidden behind card images on gravel sector cards (and KOM card 0).

Purpose: The ::before pseudo-element badge has no z-index. On cards where the image container is `position: relative` (gravel i<2, KOM i===0), both are positioned elements competing at the same stacking level within the `isolation: isolate` context. DOM order causes ::before to paint first (behind the image). Adding `z-index: 1` to the ::before guarantees badge paints above the image container.

Output: One CSS property addition that resolves the v10.4 UAT Test 1 failure.
</objective>
