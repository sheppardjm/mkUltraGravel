Create `netlify.toml` at project root with:
```toml
[build]
  command = "npm run build"
  publish = "dist"
  functions = "netlify/functions"

[functions]
  node_bundler = "esbuild"

[[redirects]]
  from = "/api/*"
  to = "/.netlify/functions/:splat"
  status = 200
```

Create `netlify/functions/strava-auth.js` using v1 handler syntax (`exports.handler`, NOT v2 `export default` -- active env var bug in v2 as of 2026-03-28):

The function:
1. Reads `activityUrl` from `event.queryStringParameters`
2. Validates activityUrl matches `/strava\.com\/activities\/\d+/` -- returns 400 if invalid
3. Generates a 16-byte hex nonce via `require('crypto').randomBytes(16).toString('hex')`
4. Builds a state payload: `JSON.stringify({ nonce, activityUrl })`
5. Base64url-encodes the state: `Buffer.from(statePayload).toString('base64url')`
6. Constructs the Strava authorization URL with params:
   - `client_id`: from `process.env.STRAVA_CLIENT_ID`
   - `redirect_uri`: from `process.env.STRAVA_REDIRECT_URI`
   - `response_type`: `'code'`
   - `approval_prompt`: `'auto'`
   - `scope`: `'activity:read_all'` (NOT just activity:read -- needed for include_all_efforts)
   - `state`: the base64url-encoded state
7. Returns 302 redirect to Strava with:
   - `Location` header: the Strava auth URL
   - `Set-Cookie` header: `strava_oauth_state=${nonce}; HttpOnly; Secure; SameSite=Lax; Max-Age=600; Path=/`

Important: Use `activity:read_all` scope, not `activity:read`. The `include_all_efforts=true` parameter in the callback requires this scope to reliably return all 9 segment efforts.
  </action>

  </verify>
