Add a `buildSegmentLeaderboard(athletes, segId)` function to the frontmatter of `results.astro`. This function:
1. Filters athletes to those who have `segments[segId]?.elapsed_time`
2. Maps to `{ name, gender, activityUrl, elapsed_time }`
3. Sorts by `elapsed_time` ascending (fastest first)
4. Returns the sorted array

Build leaderboards for all 9 segments:
```javascript
const ALL_SEGMENT_IDS = [...SECTOR_SEGMENT_IDS, ...KOM_SEGMENT_IDS];
const ALL_SEGMENT_NAMES = { ...SECTOR_NAMES, ...KOM_NAMES };
const segmentLeaderboards = {};
for (const segId of ALL_SEGMENT_IDS) {
  segmentLeaderboards[segId] = buildSegmentLeaderboard(athletes, segId);
}
```
