In `src/components/RouteMap.astro`, modify the sector rendering loop to also place a persistent star-rating badge at each sector's midpoint:

1. **Create a LayerGroup for badges** (before the sectors forEach):
   ```ts
   const sectorBadges = L.layerGroup();
   ```

2. **Inside the existing `annotations.sectors.forEach` loop**, after the existing `L.polyline(...).bindPopup(...).addTo(map)` block, add a midpoint badge:
   - Calculate the midpoint index: `const mid = Math.floor(sector.track.length / 2);`
   - Get the midpoint coords: `const midPt = sector.track[mid];`
   - Build the star string: `const starStr = '★'.repeat(sector.stars) + '☆'.repeat(5 - sector.stars);`
   - Create a divIcon marker at the midpoint:
     ```ts
     const badge = L.marker([midPt.lat, midPt.lon], {
       icon: L.divIcon({
         className: 'sector-badge',
         html: `<span style="color:${starColors[sector.stars] || '#ffffff'}">${starStr}</span>`,
         iconSize: [null, null] as any,
         iconAnchor: [0, 0]
       }),
       interactive: false
     });
     sectorBadges.addLayer(badge);
     ```
   - Use `interactive: false` so the badge doesn't intercept clicks meant for the sector polyline popup.

3. **Add zoom-dependent visibility** (after the sectors forEach loop completes):
   ```ts
   // Show sector badges only at zoom 10+
   function updateBadgeVisibility() {
     if (map.getZoom() >= 10) {
       if (!map.hasLayer(sectorBadges)) map.addLayer(sectorBadges);
     } else {
       if (map.hasLayer(sectorBadges)) map.removeLayer(sectorBadges);
     }
   }
   map.on('zoomend', updateBadgeVisibility);
   updateBadgeVisibility(); // initial check
   ```

4. **Add CSS for `.sector-badge`** in `src/styles/global.css`, inside the existing `@layer components { ... }` block (near the other Leaflet overrides):
   ```css
   :global(.sector-badge) {
     background: transparent !important;
     border: none !important;
   }
   ```
   Wait — global.css is not inside an Astro component, so `:global()` is NOT needed there. The existing `.restock-marker` rule uses `:global()` because it's inside `RouteMap.astro`'s `<style>` block. In global.css, just use plain selectors.

   Add to global.css inside `@layer components`:
   ```css
   .sector-badge {
     background: transparent !important;
     border: none !important;
   }
   .sector-badge span {
     font-size: 0.7rem;
     letter-spacing: -0.5px;
     white-space: nowrap;
     text-shadow: 0 0 3px rgba(0, 0, 0, 0.9), 0 0 6px rgba(0, 0, 0, 0.7);
     pointer-events: none;
   }
   ```
   The text-shadow ensures readability against the dark map tiles. The font-size keeps badges compact. `pointer-events: none` on the span reinforces non-interactivity.
