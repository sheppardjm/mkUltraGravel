3. Create `src/components/PhotoGallery.astro`:

   **Frontmatter:**
   - Import PhotoSwipe CSS in frontmatter (safer pattern per research — avoids deployment issues with CSS-in-script-tag): `import 'photoswipe/style.css';`
   - Read photos.json using the established build-time pattern:
     ```javascript
     import fs from 'node:fs';
     import path from 'node:path';
     const photosRaw = fs.readFileSync(path.join(process.cwd(), 'public/data/photos.json'), 'utf-8');
     const photos = JSON.parse(photosRaw);
     ```
   - This matches the pattern from GravelSectors.astro (decision [06-01]: fs.readFileSync + process.cwd())

   **Template:**
   - Render a grid container: `<div id="photo-gallery" class="grid grid-cols-2 md:grid-cols-3 lg:grid-cols-4 gap-2">`
   - Map over photos array, rendering each as a `<button>` element:
     ```html
     <button
       type="button"
       class="gallery-item aspect-[3/4] overflow-hidden cursor-pointer border-0 p-0 bg-transparent"
       data-index={i}
       data-src={`/images/${photo.filename}`}
       data-w={photo.width}
       data-h={photo.height}
       aria-label={`View route photo at mile ${photo.mi}`}
     >
       <img
         src={`/images/thumbs/${photo.filename.replace(/\.(jpg|jpeg|png)$/i, '.webp')}`}
         alt={`Route photo at mile ${photo.mi}`}
         loading={i < 8 ? 'eager' : 'lazy'}
         decoding="async"
         class="w-full h-full object-cover transition-transform duration-300 hover:scale-105"
       />
     </button>
     ```
   - Note: First 8 images (2 rows of 4-col grid) load eagerly; rest are lazy
   - Landscape images (width > height) use `aspect-[3/4]` on the button with `object-cover` on the img — the grid stays uniform, landscape images are cropped to portrait in the grid but shown full in lightbox

   **Script tag (NOT is:inline — let Astro bundle it):**
   ```html
   <script>
   import PhotoSwipeLightbox from 'photoswipe/lightbox';

   function initGallery() {
     const gallery = document.getElementById('photo-gallery');
     if (!gallery) return;

     const items = Array.from(gallery.querySelectorAll('.gallery-item')).map(el => ({
       src: el.dataset.src,
       width: parseInt(el.dataset.w, 10),
       height: parseInt(el.dataset.h, 10),
       alt: el.querySelector('img')?.alt || '',
     }));

     const lightbox = new PhotoSwipeLightbox({
       dataSource: items,
       pswpModule: () => import('photoswipe'),
       bgOpacity: 0.95,
     });
     lightbox.init();

     gallery.querySelectorAll('.gallery-item').forEach((el, i) => {
       el.addEventListener('click', () => lightbox.loadAndOpen(i));
     });
   }

   // Support Astro View Transitions
   document.addEventListener('astro:page-load', initGallery);
   // Fallback for non-View-Transitions pages
   if (!document.querySelector('[data-astro-transition]')) {
     initGallery();
   }
   </script>
   ```

   PhotoSwipe defaults handle:
   - `escKey: true` — Escape closes (default)
   - `bgClickAction: 'close'` — click outside closes (default)
   - Arrow keys for navigation (default)
   - Touch swipe for mobile (default)
   - Close button visible in UI (default)

   Do NOT use `define:vars` or `is:inline` — these kill Astro bundling (decision [07-01]).
