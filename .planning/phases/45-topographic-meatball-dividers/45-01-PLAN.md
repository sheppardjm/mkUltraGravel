  <files>src/components/TopoDivider.astro</files>
  <action>
Create `src/components/TopoDivider.astro` — a self-contained Astro component with inline SVG, scoped `<style>`, and `<script>` block.

**SVG structure:**
- `<svg>` with `class="topo-divider"`, `data-topo-divider` attribute, `aria-hidden="true"`, `xmlns`, `viewBox="0 0 200 200"`, `width="200"`, `height="200"`
- 5 concentric `<circle>` elements sharing center `cx="100" cy="100"`, with increasing radii (approximately 18, 34, 52, 70, 88)
- All circles: `fill="none"`, `stroke="currentColor"`, `stroke-width="1.5"` (outermost ring slightly thinner at 1.2), `pathLength="1"`, `stroke-dasharray="1"`, `stroke-dashoffset="1"`
- Each circle gets classes `ring ring-N` (1 through 5)

**CSS (scoped `<style>`):**
- `.topo-divider`: `display: block; margin: 0 auto; color: var(--color-accent-green); opacity: 0.5;`
- `.topo-divider .ring`: `stroke-dashoffset: 1; transition: stroke-dashoffset 1s ease-out;`
- `.topo-divider.is-drawn .ring-1` through `.ring-5`: `stroke-dashoffset: 0;` with staggered `transition-delay` (0ms, 100ms, 200ms, 300ms, 400ms)
- `@media (prefers-reduced-motion: reduce)`: `.topo-divider .ring { stroke-dashoffset: 0; transition: none; }` — rings display statically, no animation

**JavaScript (`<script>` block):**
- Define `initTopoDividers()` function
- Inside: check `window.matchMedia("(prefers-reduced-motion: reduce)").matches` — if true, return immediately (skip observer)
- Create `IntersectionObserver` with `rootMargin: "0px 0px -60px 0px"` and `threshold: 0.1`
- Observer callback: on `isIntersecting`, add `is-drawn` class to `entry.target`, then `unobserve` (one-shot)
- Query all `[data-topo-divider]` elements and observe each
- Register: `document.addEventListener("astro:page-load", initTopoDividers);`
- Fallback: `if (!document.querySelector("[data-astro-transition]")) initTopoDividers();`

This follows the exact same IntersectionObserver lifecycle pattern used in `index.astro` (revealObserver) and `RouteMap.astro`/`ElevationProfile.astro`.
