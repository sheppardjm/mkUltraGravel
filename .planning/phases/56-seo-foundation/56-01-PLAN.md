       - Add `site: "https://mkultragravel.com",` as a top-level property in `defineConfig()`
       - Add `integrations: [sitemap()],` to `defineConfig()`
       - Preserve all existing config (vite plugins, fonts)

    The final config should look like:
    ```javascript
    import { defineConfig, fontProviders } from "astro/config";
    import tailwindcss from "@tailwindcss/vite";
    import sitemap from "@astrojs/sitemap";

    export default defineConfig({
      site: "https://mkultragravel.com",
      integrations: [sitemap()],
      vite: {
        plugins: [tailwindcss()],
      },
      fonts: [
        // ... existing font config unchanged
      ],
    });
    ```

    3. Run `npm run build` and verify:
       - `dist/sitemap-index.xml` exists and contains a `<sitemap>` entry pointing to `sitemap-0.xml`

  <verify>

    - sitemap-index.xml contains `<loc>` with sitemap-0.xml URL

<task type="auto">
  <name>Task 2: Create robots.txt and add Netlify subdomain redirect</name>
  <files>public/robots.txt, netlify.toml</files>
  <action>
    1. Create `public/robots.txt` with this exact content:
       ```
       User-agent: *
       Allow: /

       Sitemap: https://mkultragravel.com/sitemap-index.xml
       ```
       Note: No trailing newline issues — file should end with a newline after the Sitemap line.

    2. Edit `netlify.toml` to add a subdomain redirect rule. Add this block BEFORE the existing `/api/*` redirect (order matters for Netlify — more specific host-based redirects should come first):
       ```toml
       [[redirects]]
         from = "https://mkultragravel.netlify.app/*"
         to = "https://mkultragravel.com/:splat"
         status = 301
         force = true
       ```
       The `force = true` is critical — without it, the redirect may not fire if the origin returns a 200.

    3. Verify after build:
       - `dist/robots.txt` exists with the correct content (Astro copies public/ files as-is)
       - `netlify.toml` has both redirect blocks (subdomain + api)
  </action>
  <verify>
    Run: `npm run build && cat dist/robots.txt && cat netlify.toml`
    - dist/robots.txt contains "User-agent: *", "Allow: /", and "Sitemap: https://mkultragravel.com/sitemap-index.xml"
    - netlify.toml contains the subdomain redirect with status 301 and force = true
  </verify>
  <done>
    robots.txt is in dist/ with Allow directive and Sitemap URL. netlify.toml has subdomain redirect with 301 + force = true. SEO-05 (deploy preview noindex) requires no configuration — Netlify adds X-Robots-Tag: noindex to deploy preview responses by default.
  </done>
</task>
