<task type="auto">
  <name>Task 2: Write parse-gpx.js and generate route-data.json</name>
  <files>scripts/parse-gpx.js, public/data/route-data.json, public/mk-ultra.gpx</files>
  <action>
Create `scripts/parse-gpx.js` that:

1. Reads `MK Ultra.gpx` from project root via `fs.readFileSync`.

2. Parses it with gpxparser. IMPORTANT: gpxparser needs DOMParser in Node.js. Try parsing first;
   if `gpx.tracks` is empty or gpxparser throws "DOMParser is not defined", add this shim at the
   top of the script BEFORE requiring gpxparser:
   ```javascript
   const { DOMParser } = require('@xmldom/xmldom');
   global.DOMParser = DOMParser;
   ```
   If the shim is needed, install @xmldom/xmldom: `npm install @xmldom/xmldom`

3. Extracts `tracks[0].points` and `tracks[0].distance.cumul`.

4. CRITICAL: Handle the cumul off-by-one. `cumul[0]` is the distance from point 0 to point 1,
   NOT zero. Build the mile array by prepending 0:
   ```javascript
   const cumulMiles = [0, ...track.distance.cumul.map(m => m / 1609.344)];
   // cumulMiles[i] now corresponds to points[i]
   ```
   Verify: `cumulMiles.length === points.length` (if cumul has length N-1, this prepend makes it N).
   If cumul already has length === points.length, then: `cumulMiles = [0, ...cumul.slice(0, -1).map(m => m / 1609.344)]`
   -- test both and use whichever produces `cumulMiles[0] === 0` and `cumulMiles.length === points.length`.

5. Maps to output format:
   ```javascript
   const routeData = points.map((pt, i) => ({
     lat: pt.lat,
     lon: pt.lon,
     ele: pt.ele,
     mi: Math.round(cumulMiles[i] * 10000) / 10000  // 4 decimal places
   }));
   ```

6. Creates `public/data/` directory (mkdir -p) and writes `public/data/route-data.json` with
   `JSON.stringify(routeData, null, 2)`.

7. Copies `MK Ultra.gpx` to `public/mk-ultra.gpx` (clean URL, no spaces) using `fs.copyFileSync`.

8. Logs summary to console:
   - Total trackpoints
   - First point: lat, lon, mi (should be 0)
   - Last point: lat, lon, mi (should be ~79.6)
   - Total distance in miles

After creating the script, run it: `node scripts/parse-gpx.js`

Validate the output:
- `route-data.json` has 1,827 entries (or close -- the research says 1,827)
- First entry has `mi: 0`
- Last entry has `mi` approximately 79.6
- All entries have non-null lat, lon, ele, mi
- `public/mk-ultra.gpx` exists and is the same size as `MK Ultra.gpx`
  </action>
