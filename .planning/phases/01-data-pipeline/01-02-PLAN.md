Step 1: Identify all candidate route photos.
- List all JPGs in `images/` (NOT `images/tone/`)
- Exclude files with ` (1).jpg` suffix (known duplicates: `HPBlbKhBz0...(1).jpg` and `y0WSG2Mc...(1).jpg`)
- This should yield approximately 48 candidates (51 total - 2 duplicates - ~12 tone images that are in the tone/ subdirectory... but tone/ has ~31 images per research, and they're in a subdirectory, so glob images/*.jpg directly)

Step 2: Visually inspect all candidate JPGs.
- Read each image file using the Read tool to visually inspect it
- Categorize each as:
  - **Route photo**: Shows the cycling route, trail, gravel road, scenery along the route, riders on the route
  - **Non-route**: Gear shots, portraits, pre/post-ride photos, unrelated images
- The target is exactly 33 route photos. If you find more or fewer, document the discrepancy.

Step 3: For each of the 33 route photos, estimate a mile marker (0-79.6).
- Use visual cues: terrain type, vegetation, road surface, landmarks
- Distribute photos roughly evenly along the route if no specific clues
- The research notes these annotation landmarks for reference:
  - Mile 21.8-23.3: Laughing Whitefish River area, Billie Helmer KOM
  - Mile 37.3-39.4: Chatham area, Akkala Rd sector starts
  - Mile 43.3: Haavisto sector
  - Mile 46.3: Rumely area
  - Mile 50.7: Forest Service Rd sector
  - Mile 58.7: C4 sector (the hardest gravel)
  - Mile 76.1: Dollar General, nearing end
  - Mile 78.1: Silver Creek KOM
- If photos show similar terrain, space them within the same general area
- Precision to 1 decimal place is fine (e.g., 15.0, 23.5, 42.0)

Step 4: Create `scripts/photo-manifest.js`:
```javascript
/**
 * Photo manifest - curated list of 33 route photos with estimated mile markers.
 * Generated by visual inspection of images/ directory.
 * Used by scripts/match-photos.js to produce public/data/photos.json.
 */
const photoManifest = [
  { filename: 'example-filename-1536x2048.jpg', mi: 5.0 },
  // ... 33 entries total
];

module.exports = { photoManifest };
```

Ensure:
- Exactly 33 entries
- All filenames match actual files in images/
- Mile markers are between 0 and 79.6
- Mile markers are in ascending order (sorted by route position)
- No duplicate filenames
