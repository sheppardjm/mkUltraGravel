  <action>
Create `scripts/resolve-annotations.js` that:

1. Reads `public/data/route-data.json` (produced by Plan 01-01). If it doesn't exist, print
   an error message and exit with code 1: "Error: route-data.json not found. Run parse-gpx.js first."

2. Implements `findPointAtMile(routeData, targetMile)`:
   - Linear scan through routeData (already sorted by mi ascending)
   - Find the point with smallest `|pt.mi - targetMile|`
   - Early exit optimization: break when `pt.mi > targetMile + 0.5`
   - Returns `{ lat, lon, actualMi }`
   - If targetMile exceeds the last trackpoint's mi, clamp to last point and log a warning:
     `console.warn('WARNING: Mile marker ${targetMile} exceeds route end (${maxMi}). Clamped to last trackpoint.')`

3. Implements `findPointsForSegment(routeData, startMi, lengthMi)`:
   - Calls findPointAtMile for startMi to get start lat/lon
   - Calls findPointAtMile for (startMi + lengthMi) to get end lat/lon
   - Also extracts all trackpoints between startMi and endMi as a `track` array
     (array of {lat, lon} for rendering the segment polyline on the map)
   - Returns `{ lat, lon, endLat, endLon, endMi, track }`

4. Hardcodes the annotation data from data.md (do NOT parse data.md -- it's free-form text,
   just hardcode the values directly):

   Sectors:
   ```javascript
   const sectors = [
     { name: 'Sandstrom', startMi: 23.3, lengthMi: 5.89, stars: 3 },
     { name: 'Akkala Rd', startMi: 39.4, lengthMi: 1.42, stars: 3 },
     { name: 'Haavisto', startMi: 43.3, lengthMi: 1.42, stars: 4 },
     { name: 'Forest Service Rd', startMi: 50.7, lengthMi: 6.45, stars: 2 },
     { name: 'C4', startMi: 58.7, lengthMi: 5.65, stars: 5 },
     { name: 'Down Jeep', startMi: 83.0, lengthMi: 0.6, stars: 5 },
   ];
   ```

   KOM segments:
   ```javascript
   const koms = [
     { name: 'Billie Helmer', startMi: 21.9, lengthMi: 0.69, grade: 6.4, elevFt: 236 },
     { name: 'Leaving Chatham', startMi: 37.5, lengthMi: 0.33, grade: 4.1, elevFt: 72 },
     { name: 'Silver Creek', startMi: 78.1, lengthMi: 1.61, grade: 4.4, elevFt: 373 },
   ];
   ```

   Restock points:
   ```javascript
   const restocks = [
     { name: 'Laughing Whitefish River', mi: 21.8 },
     { name: 'Chatham Convenience Store', mi: 37.3 },
     { name: 'Rumely Gas Station', mi: 46.3 },
     { name: 'Dollar General', mi: 76.1 },
   ];
   ```

5. Resolves each annotation:
   - For sectors: call findPointsForSegment, merge result with sector data
   - For KOMs: call findPointsForSegment, merge with KOM data
   - For restocks: call findPointAtMile, add lat/lon to restock data

6. Writes output to `public/data/annotations.json` matching this shape:
   ```json
   {
     "sectors": [
       {
         "name": "Sandstrom",
         "startMi": 23.3,
         "lengthMi": 5.89,
         "stars": 3,
         "lat": 46.xxx,
         "lon": -86.xxx,
         "endLat": 46.xxx,
         "endLon": -86.xxx,
         "endMi": 29.19,
         "track": [{"lat": 46.xxx, "lon": -86.xxx}, ...]
       }
     ],
     "kom": [...],
     "restock": [...]
   }
   ```

7. Logs summary: count of sectors/KOMs/restocks resolved, any warnings about clamped mile markers.
