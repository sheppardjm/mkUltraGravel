  <name>Task 1: Create SiteNav.astro and integrate into BaseLayout</name>
  <files>src/components/SiteNav.astro, src/layouts/BaseLayout.astro</files>
  <action>
    Create `src/components/SiteNav.astro` with the following implementation:

    **Frontmatter:**
    - Read `Astro.url.pathname` and normalize: if rawPath is "/" keep "/", otherwise strip trailing slash via `.replace(/\/$/, "")`
    - Define navLinks array: `[{href: "/", label: "Home"}, {href: "/results", label: "Results"}, {href: "/submit", label: "Submit"}]`
    - Define `isActive(href)` function: for "/submit" match both `pathname === "/submit"` and `pathname.startsWith("/submit-confirm")`; for all other links use exact `pathname === href`

    **Template:**
    - `<nav class="site-nav">` containing:
      - Brand link: `<a href="/" class="nav-brand">MK Ultra Gravel</a>`
      - Links container: `<div class="nav-links">` mapping over navLinks, each `<a>` gets `aria-current={isActive(link.href) ? "page" : undefined}` and class "nav-link"

    **Scoped styles (inside `<style>` tag):**
    - `.site-nav`: `position: fixed; top: 0; left: 0; right: 0; z-index: 10000; display: flex; align-items: center; justify-content: space-between; padding: 0.75rem 1.5rem; background-color: oklch(0.10 0.01 250 / 0.92); backdrop-filter: blur(8px); border-bottom: 1px solid var(--color-border); font-family: var(--font-mono);`
    - `.nav-brand`: `font-family: var(--font-display); color: var(--color-accent-white); text-decoration: none; font-size: 1rem; letter-spacing: 0.05em; text-transform: uppercase;`
    - `.nav-links`: `display: flex; gap: 1.5rem;`
    - `.nav-link`: `color: var(--color-text-muted); text-decoration: none; font-size: 0.8rem; letter-spacing: 0.08em; text-transform: uppercase; transition: color 0.15s ease;`
    - `.nav-link:hover`: `color: var(--color-accent-white);`
    - `.nav-link[aria-current="page"]`: `color: var(--color-accent-green); border-bottom: 2px solid var(--color-accent-green); padding-bottom: 2px;`

    **Modify `src/layouts/BaseLayout.astro`:**
    - Add import: `import SiteNav from "../components/SiteNav.astro";` in the frontmatter
    - Add `class="pt-12"` to the `<body>` tag (currently has no class)
    - Add `<SiteNav />` as the first child of `<body>`, before the grain-overlay div
  </action>
