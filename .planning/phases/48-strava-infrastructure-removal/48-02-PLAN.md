  <action>
**package.json edits:**

1. Remove the `"validate"` script line:
   ```
   "validate": "node scripts/validate-results.mjs"
   ```

2. Remove the `"test"` script line:
   ```
   "test": "vitest run"
   ```
   Reason: scoring.test.js was the only test file. With it deleted, `vitest run` exits non-zero (no test files found), which would break CI if anyone runs `npm test`.

3. After removing both, ensure the remaining scripts block has correct comma placement (no trailing comma on the last script entry).

**netlify.toml edits:**

1. Remove the `functions = "netlify/functions"` line from the `[build]` section.

2. Remove the entire `[functions]` block:
   ```toml
   [functions]
     node_bundler = "esbuild"
   ```

3. Keep the `[[redirects]]` block intact — the /api/* redirect is a harmless no-op (per out-of-scope decision in requirements) and doesn't need to be removed.

4. Keep the `[build.environment]` block intact — SECRETS_SCAN_OMIT_PATHS and SECRETS_SCAN_OMIT_KEYS are still relevant for the .planning/ directory.

Final netlify.toml should look like:
```toml
[build]
  command = "npm run build"
  publish = "dist"

[build.environment]
  SECRETS_SCAN_OMIT_PATHS = ".planning/"
  SECRETS_SCAN_OMIT_KEYS = "GITHUB_REPO,GITHUB_OWNER"

[[redirects]]
  from = "/api/*"
  to = "/.netlify/functions/:splat"
  status = 200
```
