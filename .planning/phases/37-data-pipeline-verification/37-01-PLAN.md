Step 3 — POST the crafted payload to the deployed production function:

```bash
curl -s -w "\n\nHTTP_STATUS: %{http_code}\n" \
  -X POST "https://mkultragravel.netlify.app/.netlify/functions/submit-result" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  --data-urlencode "data=$DATA" \
  --data-urlencode "gender=M" \
  --data-urlencode "consent=yes"
```

Expected: HTTP 200 response with HTML containing "Results Submitted!" and athlete name "Test Pipeline".

IMPORTANT: Use `application/x-www-form-urlencoded` content type with `--data-urlencode`. The function uses `URLSearchParams` to parse — JSON body will fail with 400.
