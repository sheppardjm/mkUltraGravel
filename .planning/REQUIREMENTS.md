# Requirements: MK Ultra Gravel v10.4

**Defined:** 2026-04-08
**Core Value:** Get gravel cyclists excited enough about this ride to show up on June 7, 2026.

## v10.4 Requirements

### Card Display

- [ ] **CARD-02**: Classified badge `::before` pseudo-element renders visibly on all segment cards (no occlusion by card content or parent overflow)
- [ ] **CARD-03**: Card photo crops generated at resolution sufficient for large-screen display (gravel cards span ~900px+ on wide viewports)
- [ ] **CARD-04**: Gravel sector cards constrained to a reasonable max-width on viewports wider than 1280px

### Visual Overlays

- [ ] **VIS-16**: Escher background overlay opacity/brightness reduced so light text maintains readable contrast

### Gallery

- [ ] **GAL-02**: Horizontal masonry gallery fills columns with minimal empty space across variable photo aspect ratios

## Out of Scope

| Feature | Reason |

| New photo additions | No new photos this milestone |
| Mobile-specific gallery changes | Focus is large-screen polish |
